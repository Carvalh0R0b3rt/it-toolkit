# A simple Lovable project

This project is simply a test to see the extent and structure one can achieve using A.I. tools to develop and structure a dinamic dashboard and host it in a free environment.
It is also an experiment in pair programming using myself as a navigator and the AIs as drivers.

I have noticed some limitations in the free versions of certain AI and so I used mostly Gemini (Pro), Claude and Codex to solve certain issues in code as to not run out of lovable credits.

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
